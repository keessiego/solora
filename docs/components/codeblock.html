Een macOS-stijl window voor code snippets, inclusief copy-functionaliteit.

function welcome() {
  console.log("Welkom bij Solora UI!");
}

Interactie

  • Rood: Sluit het blok.
  • Oranje: Collapse de content.
  • Groen: Fullscreen modus.
  • Sleepbaar: Voeg de class draggable toe.
